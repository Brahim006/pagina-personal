Java a Fondo de Pablo Sznajdleder

Java a Fondo

Un libro recomendadísimo para aquellos que quieran aprender Java y el desarrollo de aplicaciones, diseño de programas con un enfoque orientado a objetos y patrones de diseño.

Link a mercadolibre (4° Edición)

Se trata de, tal vez, uno de los libros más completos en lo que respecta a Java (por lo menos hasta la SE 8). Si bien el libro tiene un enfoque predominantemente práctico, esto no quiere decir que no sea profundo, ya que en realidad engloba una gran cantidad de temas siendo un libro que te puede llevar desde un programador inicial a uno con un conocimiento bastante fuerte en el uso del lenguaje. Si bien no tiene ejercícios, cuenta con bastante código por el cual desarrolla todos los temas tratados, el cual recomiendo analizar, probar y comentar para tener un entendimiento completo. Además no sólo intenta enseñarnos las bases del lenguaje, si no que tembién toma capítulos enteros para aplicar el conocimiento en el desarrollo de aplicaciones reales sin escatimar en detenerse en conceptos como patrones de diseño y buenas prácticas para llevarnos a adquirir un fuerte punto de vista como desarrolladores de aplicaciones. Como cereza del pastel, en sus últimos capítulos incluye una introducción a Spring y Hibernate, dos de los frameworks más usados en la actualidad (no sin antes detenerse a explicarnos qué es un framework y detenerse a desarrollar y probar algunos). Personalmente, me gusta el enfoque que tiene este libro ya que como dije no se detiene sólo en enseñarnos el lenguaje, si no que va más allá enseñándonos a programar.

Desarrollo web con Java desde cero

Es un libro corto y sencillo (y barato, por qué no) que habla del desarrollo web con Java. Si bien no lo terminé todavía, lo encuentro bastante interesante ya que se detiene en algunos conceptos de la programación de aplicaciones web que no he visto en otros recursos que hablen sobre el tema. No estoy muy seguro de qué tan actualizado está en el tema ya que tienen unos cuantos años, pero parece ser bueno para los que quieran hacer "algo nuevo" en Java. Cuenta con algunos ejercícios y código bastante claro que ayudan a adentrarse bien en sus conceptos, siendo simple pero conciso. Una cosa a tener en cuenta es que antes de poder usarlo se requieren conocimientos digamos que "intermedios" de Java (funcionamiento de servidores, diseños de aplicaciones orientado a objetos, uso de frameworks e interfaces y flujos de datos), además de tener un dominio aunque sea básico de HTML.

Desarrollo web con Java desde Cero

El libro se presenta como complemento a Java desde Cero de la misma editorial, cuenta con varios ejemplos integradores además de preguntas y ejercícios que invitan a re-leer y profundizar sobre los contenidos de cada capítulo.

HTML, CSS y JavaScript de José Luján

HTML, CSS y JavaScript

Link a Mercado Libre

Es un libro sobre el que tengo sentimientos encontrados: por un lado lo encuentro demasiado básico, que repite mucho código y no suele profundizar mucho. Sin embargo, reconozco que es extremadamente amigable para introducirse a estos tres lenguajes y cómo se relacionan entre sí, contando con varias imágenes ilustrativas, texto conciso y bien separado por temas. Además, puede servir como material de consulta sobre algunos temas básicos.

Android Studio - Aprende a desarrollar aplicaciones

Este libro, del mismo autor que el anterior (José Luján), tiene la misma filosofía pero orientada al deasarrollo de aplicaciones nativas en Android con Java. Es un buen punto de introducción, además de que habla sobre conceptos básicos de programación orientada a objetos antes de comenzar a programar la interactividad en Java. Por lo demás, no escatima en detenerse a explicar conceptos propios del desarrollo móvil, como lo son distintos tamaños y densidades de pantalla, actividades y acceso a los dstintos sensores, además de un fuerte manejo del IDE de Android Estudio.

Lo conocí cuando estaba haciendo mi curso de Diseño Web y terminó siendo mi referencia para consultas cuando estoy desarrollando algún proyecto con tecnologías como HTML y CSS. Font Awesome

Font Awesome es una plataforma que provée de íconos para incluir en nuestras páginas de una forma sencilla y sin otro requerimiento que registrarse en la página. Si bien ofrece un servicio premium, la versión gratuita de por sí ofrece una cantidad importante de íconos de todo tipo. Su uso es sencillo, sólo requiere la inclución de una línea de script dentro del head de nuestro documento HTML y a partir de ahí se pueden empezar a incluir los íconos como etiquetas que serán tratadas como texto por el navegador (por lo menos en lo que a coloreado y tamaño respecta). Todos los íconos incluidos en este sitio provienen de este servicio.

Visual Studio Code

Se trata del IDE que más uso para desarrollo web (ya que para proyectos en Java uso mayormente NetBeans). Lo más destacable desde mi punto de vista es la cantidad de lenguajes que usa y la forma sencilla en la que se maneja, haciéndolo ideal para proyectos en los que se combinan distintos lenguejes y tecnologías. Además, se integra perfectamente con GitBash haciendo el manejo de Git bastante cómodo y trivial, ya que cuenta con su propia terminal para ejecutar comandos y la forma de manejar los "stages" y "commits" es bastante intuitiva.
