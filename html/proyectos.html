Desarrollo web

Página de We Colour estilistas

Se trata del proyecto final del curso de Diseño web brindado por Coderhouse. Es una página relativamente sencilla orientada a cubrir el modelo de negocios y dar a conocer distinta información sobre servicios y contacto de la peluquería We Colour Estilistas, situada en la ciudad de Córdoba, Argentina.

Su desarrollo fue lento, ya que obviamente fue el lienzo sobre el que fui plasmando todo lo aprendido durante el curso, sin embargo intenté dejarla a un nivel más manejable (hablando del código), en caso de que se requieran hacer posteriores modificaciones. Ya que después de todo, la idea es que sea una página funcional para complementar la oferta de comunicación y publicidad de la peluquería.

Repositorio GitHub

En esta página d mis primeros pasos en dos herramientas que son las bases del desarrollo web: HTML y CSS. El punto de inflexión en su desarrollo creo yo fue cuando empecé a implementar SCSS (SASS, para ser más específico), ya que mejoró enormemente la la forma de organizar el código CSS, hacíendolo mucho más fácil de leer y modificar. Sin duda, una de las cosas que más me costó asimilar fue la correcta separación en media queries para los diseños responsivos, por lo que decidí crear diseños para (además de una versión de escritorio) crear una versión móvil y una versión para tablets, tratando de mantener la coherencia entre todas las versiones. En la última etapa de desarrollo, decidí incluir un poco de código PHP para el envío de la información que se escriba en el formulario de contacto. Además, decidí dar mi primer paso en JavaScript, creando un archivo para poder animar un carrusel de imágenes el cual se encuentra en la sección de contacto. Tabién es mi primer proyecto gestionado a través de Git y Github.

Qué me enseñó este proyecto:

- Toda la base del desarrollo web, desde las tecnologías bases hasta el proceso pre-diseño.
- Una visión del desarrollo de software apuntada a un perfil más comercial.
- Una probada al leguaje de programación de JavaScript, que despertó mi interés para seguir profundizando en él.
- Herramientas de control de versiones, tales como Git y GitHub
